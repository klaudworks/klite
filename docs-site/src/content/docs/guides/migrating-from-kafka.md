---
title: Migrating from / to Kafka
description: Migrate between Apache Kafka and klite using MirrorMaker 2.
---

klite speaks the standard Kafka wire protocol, so you can use [MirrorMaker 2](https://kafka.apache.org/documentation/#georeplication-mirrormaker2) (MM2) to migrate data from Kafka to klite or from klite back to Kafka. MM2 continuously replicates topics and consumer group offsets between clusters with zero downtime.
